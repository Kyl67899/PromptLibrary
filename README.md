# PromptLibrary

PromptLibrary is a community-driven prompt collection for building frontend features using AI-assisted prompts. It helps developers explore UI patterns, customize component behavior, and build new layouts faster with reusable prompt templates.

## Features

- Prompt library for frontend UI patterns and component designs
- Modern Next.js application structure with `app/`, `components/`, and `lib/`
- Admin API endpoints for campaigns, prompts, and newsletter workflows
- Ready for extension and community contributions

## Getting Started

### Requirements

- Node.js 18+ or later
- npm or compatible package manager

### Install

```bash
npm install
```

### Run the app locally

```bash
npm run dev
```

Open `http://localhost:3000` in your browser to view the app.

## Project Structure

- `app/` - application routes, pages, and API endpoints
- `components/` - reusable UI components and shared interface elements
- `lib/` - utilities, stores, and data helpers
- `public/` - static assets and preview files
- `styles/` - global and shared styles
